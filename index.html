Beginner Friendly
1

Download and Extract

Secure the stable build package GridStorage_Server.zip, and extract assets directly to a folder on your computer.

2

Run the Startup Script

Launch the background service scripts designed specifically for your OS:

Mac / Linux bash start_server.sh

Connect Your App

🛠️ OPTION B: Advanced Setup

Power Users
1

Secure Your Node

Extract files, and open configuration settings inside a text editor to update default security keys:

docker-compose.yml YAML
# Set your personal secret keys
POSTGRES_PASSWORD: "strong_postgres_password"
PGRST_JWT_SECRET: "your-custom-jwt-secret-min-32-chars"
2

Start the Server Manually

Open your terminal inside the server workspace block and release the docker orchestrator:

docker-compose up -d EXECUTE

🛑 Troubleshooting & FAQ

"Server is Unreachable"

Verify your phone and primary server machine are on the exact same local router space. If on Windows, ensure Windows Defender Firewall allows raw inbound requests through TCP Port 3000.
