OPTION A: Quick Start

Beginner
1

Download and Extract

Secure the stable build package and extract assets directly to a clean folder.

2

Run the Startup Script

Boot the automation services mapped for your OS:

Windows OS Double-click start_server.bat
Mac / Linux bash start_server.sh
3

Connect Your App

The script will print your Local IP Address in the terminal. Enter this in your GridStorage app.

OPTION B: Advanced Setup

Power Users

Open configuration files in a text editor to update default security keys:

docker-compose.yml
# Set your personal secret keys
POSTGRES_PASSWORD: "your_password"
PGRST_JWT_SECRET: "your_32_char_secret_key"

Open your terminal inside the server folder and run:

docker-compose up -d
